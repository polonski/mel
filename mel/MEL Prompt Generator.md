# MEL Prompt Generator

The **MEL Prompt Generator** is a VS Code extension that provides an interactive Webview UI for generating MEL prompts. It integrates directly with [[Gemini Code Assist]] via the VS Code chat view.

## Features
- Interactive UI for generating prompts.
- Direct integration to push prompts to the chat view.

## Developer Actions
- The extension can be published to the VS Code Marketplace using `vsce publish`.
- It can be packaged locally for sharing using `vsce package`.
